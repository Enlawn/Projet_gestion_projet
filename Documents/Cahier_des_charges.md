On a choisi le jeu qu'on allait réaliser : Ordinateur, tu utiliseras
Principe du jeu : apprendre aux joueurs à utiliser un ordinateur, sous windows (technophile et technophobe) et sous GNU Linus (technophile)

On doit donc respecter certaines contraintes 
- Le temps : on doit finir le projet d'ici Janvier
- Le contenu : on doit produire un ensemble de documents et de content pour le jeu
- Présentation orale : diapo n co
- Livrable final : Le dépot GitHub avec tout les élements contenant le jeu, maquette/ concept / code/ etc. présent à l'interieur

  On a décidé de faire le jeu Dans un site internet :
  - site fonctionnel
  - site facile d'utilisation
  - site capable de tourner en local -> l'heberger sur GitHub pour que les utilisateurs puissent récupèrer tout le contenu et le faire tourner sur leur machine

  Les scénarios des missions :
  - Divers niveau de difficulté
  - Diverse forme de mission
  - Contenu des missions varié
  - Claire / pas trop long / 1 mission = 1 élement
  - Documenter les scénarios => si possible rediriger vers des vidéos/ sites internet allant plus en profondeur sur un sujet
