Plusieurs scénario, permettant d'expliquer le plus grand spectre d'utilisation possible d'un ordinateur sous windows et linux (gnu linux)  

Sous windows :  
on va commencer par la base des bases, c'est à dire présenter les différents élements qu'on a lorsqu'on allume un ordinateur (on ne précisera pas la partie initiailisation lors du 1er allumage, des vidéos détaillées existe et l'acquisition des captures d'écrans pourrait être compliqué à réalilser)  
- présenter le menu démarrer  
- présenter l'explorateur de fichier  
- présenter la fenetre paramètres  
- présenter un moteur de recherche (celui installé par défaut, microsoft edge)  

  et ensuite présenter des cas précis :  
  - créer un fichier / créer un dossier / créer un raccourci / 
  - personnaliser son ordinateur  
  - - fond d'écran  
    - police d'affichage  
    - taille du texte / des icones  
    - accessibilité (loupe, assistance orale)  
    - couleurs de l'interface  
    - langue d'affichage  
    - heure / date
  
  - modifier les paramètres (basique et avancé avec un aperçu du paramètrage du BIOS)  
  - - paramètrer la langue de saisie  
    - paramètrer les mises à jours / mettre à jours ces drivers (pilotes) / retourner à une version précedente  
    - paramètrer la mise en veille / l'extinction / le démarrage  
    - paramètrer le mode d'alimentation
  
  - récupèrer des informations sur sa machine (userdiag)  
  - - pourquoi ?  
    - résolution de bug  
    - résolution de soucis d'optimisation  
    - résolution de driver vetuste
  
  - spécifier les performances de sa machine (cinebench, memory test n co)  
  - - pourquoi ?  
    - améliorer son set up  
    - connaître les limites de son set up  
    - adapter son utilisation
  
  - faire une recherche internet
