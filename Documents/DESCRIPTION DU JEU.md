# Notre jeu sérieux en quelques mots : 

## Un tutotriel "ludique" sur la prise en main des OS Windows (11 en particulier, vu qu'on nous force la main à aller vers lui) et présentation des lignes de commande Linux (distribution choisi par randame).  
### 2 approches : 

- **Lignes de commande sur Linux**
- **Présentation de fonctionnalités de base sur Windows** : explorateur de fichier, navigateur web, paramètres
- **Présentation de fonctionnalités avancées sur Windows** : Bios, Benchmarking, diagnostique de l'ordinateur et **lignes de commande**

### Comment ? 

- une partie cours / présentations / explications
- une partie quizz pour les notions Windows
- pour les lignes de commandes, en Linux et en Windows, une interface interactive utilisant Grub
